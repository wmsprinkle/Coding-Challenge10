# UrbanStyle Data Cleaning & Preparation Project

This project focuses on cleaning and transforming messy customer data using Python and Pandas. The goal is to prepare raw business data for analysis by addressing common real-world data issues.

## Overview
The dataset simulates a customer relationship management (CRM) system for UrbanStyle, a retail fashion company. The project applies data cleaning techniques to handle inconsistencies, missing values, and formatting issues.

## Objectives
- Identify and handle missing data
- Convert and correct data types
- Standardize text and formatting
- Remove duplicate records
- Create derived features for analysis
- Prepare a clean dataset for business use

## Tools & Technologies
- Python
- Pandas
- NumPy

## Key Features
- Missing value handling using fill and drop strategies
- Data type conversions (dates, numeric values)
- Text standardization (names, categories, formatting)
- Duplicate detection and removal
- Feature engineering (purchase behavior metrics)
- Data sorting and structuring for analysis

## Results
The final dataset is clean, consistent, and ready for analysis. Key business metrics such as customer value, purchase frequency, and category performance are derived from the cleaned data.

## Purpose
This project demonstrates the ability to clean and structure raw business data, a critical step in any data analytics workflow.
